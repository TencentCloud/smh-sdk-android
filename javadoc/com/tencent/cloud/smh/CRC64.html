Class CRC64

  • All Implemented Interfaces:
    java.util.zip.Checksum 
    
public class CRC64
 implements Checksum

    CRC-64 implementation with ability to combine checksums calculated over different blocks of data. This is a faster version of the original implementation by R. Nikitchenko, incorporating the nested lookup table design by Mark Adler (see Stackoverflow). Throughput rose from 375 MB/s to 1150 MB/s on a 2,3 GHz i7, which is 3.06 times faster.
